## Notes

### Systemic pass summary (5 accounts: dannon, kraft-heinz, dhl-supply-chain, frito-lay, kimberly-clark)

Five P0 systemic findings apply uniformly across all 40 accounts:

1. **Empty right gutter (P0)** — The `<aside aria-hidden="true">` at `memo-shell.tsx:232` is empty on every account. The 14rem right column in the `lg:grid-cols-[11rem_minmax(0,36rem)_14rem]` grid contributes nothing. The MemoAside for caveat content renders inside the article column rather than projecting into the gutter (see marginalia finding). Highest-impact systemic fix.

2. **Detention counter widget (P0)** — A floating badge showing a live-accumulating "modeled detention accrued while reading" dollar value overlays the bottom-right of the cover on every account. It is outside the approved color budget, antithetical to the anti-sales memo register, and produces implausible values for large-network accounts (DHL: $427.78; Frito-Lay: $81.99; Dannon: $6.75). Remove from all `/for/[account]` routes.

3. **Plain H1 — no accent italic phrase (P0)** — The H1 `title` prop is `string` throughout. The approved accent budget location "italicized phrase in H1" cannot be expressed. All 40 covers miss the key typographic entry point for the accent color.

4. **Running header visible on page load (P0)** — The sticky running header renders at y=0 on initial load, overlapping the cover's classification bar. On mobile it overflows the viewport at 375px (title text extends to x=416). The JavaScript opacity-transition correctly hides it visually but the element is present in the accessibility tree from first render.

5. **Evidence density: zero artifact treatments (P0)** — No account shows a redacted-artifact block (Module Inspector screenshot, redacted manifest, attribution-free quote image). The spec mandates ≥1 per memo. This is a hard fail on evidence density for all 40 accounts.

Per-account variation found:
- **Cover DL wrapping** is severity-graded by `contextDetail` length. DHL (181px) and Kimberly-Clark (114px) are the worst offenders. Dannon (69px) and Frito-Lay (69px) are borderline.
- **Hypothesis single-block prose** is worst on Kraft Heinz and Kimberly-Clark (longest single paragraphs).
- **Detention counter magnitude** scales with facility count: DHL ($427.78) > Frito-Lay ($81.99) > Kraft Heinz ($11.00) > Dannon ($6.75) > Kimberly-Clark ($5.04).
